Verify-On-Chain 🔗
> A decentralized-inspired system to verify certificates and prevent fraud.

🌐 Live Demo: https://verify-on-chain.preview.emergentagent.com/

📌 Project Overview

Verify-On-Chain is a blockchain-based certificate verification system that helps prevent fake certificates.

🚀 Features

- Admin can upload certificates
- Each certificate gets a unique ID
- Users can verify certificates using the ID
- Tamper-proof verification concept

🛠 Tech Stack

- Frontend: React
- Backend: FastAPI (Python)
- Database: MongoDB
- Blockchain: Simulated
- Storage: Simulated IPFS

🔍 How It Works

Each certificate is assigned a unique ID and stored securely.
Users can verify authenticity by entering the certificate ID.

📈 Future Improvements

- Real blockchain (Ethereum)
- IPFS integration
- QR code verification

## ▶️ How to Use

1. Open the live demo link
2. Login as admin and upload a certificate
3. Copy the generated certificate ID
4. Go to verification page
5. Enter the ID to verify authenticity
